# kanbanBoard
Сделал простую канбан-доску на Python. Минималистчино, с drag-and-drop систмой для удобства

Для базы данных используется sqlite, автоматическое создание(при первом запуске)\сохранение после изменений в файл kanban.db

Программа написана на Flask. Он запускает локальный сервер на компьютере.
После запуска программы она автоматически запускает http://127.0.0.1:5000/

для работоспособности программы необходимо установить зависимости:
pip install flask
