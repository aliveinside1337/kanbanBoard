# kanbanBoard
Сделал простую канбан-доску на Python. Минималистчино, с drag-and-drop системой для удобства

Для базы данных используется sqlite, автоматическое создание(при первом запуске)\сохранении после изменений в файл kanban.db

Программа написана при помощи Flask. Он запускает локальный сервер на компьютере.
После запуска программы она автоматически открывает http://127.0.0.1:5000/
